Where the chokepoints are.

This dashboard maps the global supply chain of thirteen critical minerals across three stages — extraction, processing, and trade — between 2000 and 2023. The matrix below shows, for each mineral and each stage, how concentrated production is in its leading country in the most recent year of available data. Hatched cells indicate stages for which public country-level data is not currently in the dataset; in many cases, this is because such data is not produced by the standard public sources. Methodology & sources

Context-aware notes

About this dashboard

This is a working v1 built for the CriMinGeo project (IAE-CSIC). It is a descriptive panorama, not a forecasting tool. Selections in the main view drive the contextual notes shown here.

Time window

All series are shown for 2000–2023. This window is set by UN Comtrade availability rather than by mining data: BGS series go back to 1971 but trade data is reliable from 2000 onwards. The full BGS history is preserved in the underlying repository.

Production data — BGS

Country-level production figures come from the British Geological Survey World Mineral Statistics. Within each commodity file, the column bgs_commodity_trans distinguishes stages (e.g. "cobalt, mine" vs "cobalt, refined"). The build script classifies these into two canonical stages — mining and refining — using keyword matching. Rows whose stage cannot be determined are dropped.

Source: bgs.ac.uk / World Mineral Statistics dataset, accessed for this build.

Trade data — UN Comtrade

Bilateral trade flows by HS code, downloaded via UN Comtrade API. Each mineral file carries a stage column ("primary" or "refined") set by the upstream download script based on which HS code the row belongs to. Values shown are USD (primaryValue); bilateral flows only — rows where the partner is "World" are excluded.

Caveat: HS codes are an imperfect proxy for the mining/refining distinction. Some stages are reported under codes that lump multiple stages together; some products move under codes that don't cleanly correspond to either category. See per-mineral notes when a profile is selected.

Source: comtrade.un.org via official API.

Reserves

Reserves estimates from USGS Mineral Commodity Summaries 2025 are present in the underlying data but not yet integrated into the dashboard view; this is a deliberate v1 scoping choice and a slot is reserved for adding the reserves-vs-production comparison in a later iteration.

Concentration metrics

Top-1 share is the percentage of global production accounted for by the leading country in a given year. HHI (Herfindahl-Hirschman Index) is the sum of squared shares (each in percentage points), ranging from near 0 (perfect competition) to 10,000 (single-country monopoly). HHI above 2,500 is conventionally read as "highly concentrated"; above 5,000 indicates extreme concentration.

Country names

Source files use different country naming conventions (BGS uses "Congo, Democratic Republic"; Comtrade uses "Dem. Rep. of the Congo"; USGS MCS uses "Congo (Kinshasa)"). The build script harmonizes these to a single canonical short name. The harmonization map covers the top producers cleanly; the long tail of small producers may show inconsistencies and is being expanded iteratively.

Build & reproducibility

Generated by code/build_dashboard_data.py from raw CSVs in data/source/. To rebuild after new data lands: replace the source files, re-run the script, commit the regenerated docs/data/*.json files. No server required — the dashboard is a static page.
